# IEA PVPS drops

IEA Photovoltaic Power Systems Programme — *Trends in Photovoltaic Applications* (annual, back to ~1992). Best public back-series for solar PV capacity, cell/module production, and installations.

File per year: `trends_YYYY.json`.

URL: https://iea-pvps.org/trends_reports/

Coverage: full IEA PVPS reporting countries (overlaps all 20 in our scope). Priority `5`.
