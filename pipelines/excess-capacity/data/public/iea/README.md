# IEA drops

Extracted from free IEA publications:

- `renewables_YYYY.json` — module mfg capacity, shipments, installations (solar & wind)
- `global_ev_outlook_YYYY.json` — battery cell capacity, EV production capacity, EV sales by country
- `electricity_YYYY.json` — installed capacity additions cross-check

Prefer PDF tables with page numbers in the `source` string so readers can verify. Use `priority: 5` (higher than IRENA's `0`, lower than paywalled BNEF's `10`).

Report URLs (all free):

- *Renewables 20XX*: https://www.iea.org/reports/renewables-20XX
- *Global EV Outlook 20XX*: https://www.iea.org/reports/global-ev-outlook-20XX
- IEA PVPS *Trends*: https://iea-pvps.org/trends_reports/
